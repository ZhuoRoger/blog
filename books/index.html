书单杂记,陆续补充（2016-08-09更新)

DataBase

  • 《数据库系统概念(原书第6版)》
  • 《数据库系统概论(第4版)》大学教材，国内难得经典
  • 《Relational Database Index Design and the Optimizers》 索引设计经典的书
  • 《Transaction Processing concepts and techniques》 InnoDB原型参考
  • 《DBA修炼之道：数据库管理员的第一本书》
  • 《DBA Survivor:Become a Rock Star DBA》 DBA的思想类书，谈的技术不多

MySQL

  • 《MySQL高性能(第3版)》
  • 《MySQL高可用(第2版)》
  • 《MySQL Troubleshooting》
  • 《MySQL技术内幕InnoDB存储引擎(第2版)》
  • 《Expert MySQL》

Redis

  • 《Redis in Action》
  • 《Redis设计与实现》

MongoDB

  • 《MongoDB权威指南(第2版)》
  • 《MongoDB in Action 2nd》
  • 《MongoDB Applied Design Patterns》

PostgreSQL

  • 《PostgreSQL修炼之道：从小工到专家》
  • 《PostgreSQL服务器编程》

计算机基础

  • 《UNIX环境高级编程（第3版）》
  • 《TCP/IP详解卷1》
  • 《Systems Performance: Enterprise and the Cloud》 中文版翻译也不错。
  • 《算法导论》
  • 《深入理解计算机系统》 看不懂两章，数学。。。
  • 《鸟哥的Linux私房菜》

SRE

  • 《Site Reliability Engineering: How Google Runs Production Systems》 中文版本快出了
  • 《深入理解Nginx:模块开发与架构解析》
  • 《HTTP权威指南》
  • 《Web容量规划的艺术》
  • 《架构即未来:现代企业可扩展的Web架构、流程和组织(原书第2版)》 冷门好书

运维开发小白

  • 《Go程序设计语言》
  • 《Python编程》

管理类

自我管理，时间管理和管理入门

  • 《Getting Things Done》 GTD，推荐软件Doit.im
  • 《奇特的一生》
  • 《高效能人士的七个习惯》
  • 《把时间当作朋友 3》
  • 《番茄工作法图解》
  • 《时间管理—给系统管理员》 有意思的书
  • 《精力管理》
  • 《习惯的力量》

管理入门

  • 德鲁克 《卓有成效的管理者》
  • 科特 《权力与影响力》
  • 马斯洛 《动机与人格》
  • 阿吉里斯 《个性与组织》 买不到了~~~
